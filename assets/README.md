# Assets

Place optional game assets here.

- **BGM:** Add `bgm.ogg` (or `bgm.mp3`) and wire it in game – see [DEVELOPMENT.md](../docs/DEVELOPMENT.md) → Audio → How to add BGM.
- **Sprites:** When ready, add per-entity PNGs (e.g. `tower_coder.png`, `auditor.png`) and follow the Art Pipeline in DEVELOPMENT.md.
